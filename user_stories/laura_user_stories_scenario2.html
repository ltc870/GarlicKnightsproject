User Stories for Scenario 2

As a non-technical user, I need a simple interface so that I don't have difficulty trying to use the program.

As a non-technical user, I don’t want to be intimidated by the program so that I feel that I would want to use it again.

As a non-technical user, I want the program to be self-explanatory so that I can find what I need.

As a non-technical user, I want to be able to find books that are not necessarily new in the market so that I can find a classic book if I want to.

As a non-technical user, I want to be able to recommend the program to others who might also be non-technical so that I can share how useful I find it.

As an older, non-technical user, I don’t want the program to be too hard to see and read so that I don't struggle to navigate through the program.

As a non-technical user, I want to see links where I can purchase a book that I am interested in so that I can easily find what I need.

As a non-technical user, I need the program to have a way to save my results so that I may access them at a later time.
