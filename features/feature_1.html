OverBook Search Feature

Description

A search feature that allows the user to enter search factors such as genre, author, publication date or subject.

As a user, I want to be able to search for books by genre.

As a user, I would like to search for books based on subject.

Constraints

The database may not be all encompassing to include all information for all existing books but will return what is available.
