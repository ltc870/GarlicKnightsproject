Morrie's User Stories 2

Morrie's 2nd set of User Stories

As a user, I want to be able to collaborate with others, like coworkers or my team.

As a user, I want to be able to organize my notes and annotations.

As a user, I want to be able to search for books by price.

As a user, I want to be able to highlight selections.

As a user, I want to be able to share my finds with friends and family.

As a user, I want to be able to share my finds to social media.

As a user, I want to be able to connect with other readers through the app.

As a user, I want to be able to annotate and take notes on what I'm reading.
