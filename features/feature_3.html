Available Format Options for Books Feature

Description

The user will be able to see what format is available for certain books or magazines. These options include a physical copy, eBook or audiobook.

As a user, I want to be able to search for only ebooks.

As a reader, I want to be able to find book recommendations that have audiobook components.

Constraints

If a user would like a specific format for a book, it may not be available in the preferred format. It will display what options are available.

Comments

Based on Sandra's and Daniel's scenarios
