User Stories for Scenario 3

As a new reader, I want to find suggestions even if I donâ€™t have any particular reading interests so that I can narrow down my search.

As a new reader, I want to be able to read short summaries of the book recommendations so that I can determine if it is something I might be interested in.

As a new reader, I want to see what books are popular among readers so that I can see what people are currently interested in.

As a new reader, I want to see reviews about a book so that I can see if others liked the book or not.

As a new reader, I want to see what other people have read who have similar interests so that it can help me figure out what types of books I might like.

As a new reader, I want to know what difficulty level certain books might be to read so that I don't waste my time on a book that may be too difficult for me to want to keep up with.

As a new reader, I want to be able to have a way mark a book on my recommendation results as like or dislike so that I can keep track of what book I might choose.

As a new reader, I want to have a way to share with others, if I find a recommendation that someone I know might like so that I can easily send it to them.
