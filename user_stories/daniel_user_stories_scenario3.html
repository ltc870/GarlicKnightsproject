Daniel's User Stories

Daniel's 3rd set of User Stories

As a user, I want to be able to create a reading list.

As a user, I want to be able edit my reading list.

As a user, I want to be able to delete books from my reading list.

As a user, I want to be able to add books easily to my reading list.

As a user, I want to be able to login with another common login such as gmail or facebook.

As a user, I want to be able to filter my finds by subject.

As a user, I want to be able to filter my finds by genre.

As a user, I want to be able to login to save my items.
