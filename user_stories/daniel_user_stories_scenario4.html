Daniel's 4th set of User Stories

As a user, I want to be able to easily submit feedback to developers.

As a user, I want to be able to easily contact the development team.

As a user, I want to be able to submit my feedback via a survey.

As a user, I want to be able to add books easily to my reading list.

As a user, I want to easily download Kindle books via OverBook.

As a user, I want to be able to easily purchase eBooks via OverBook.

As a user, I want to be able to have a button that easily direct me to the feedback page wherever I am in the website.
