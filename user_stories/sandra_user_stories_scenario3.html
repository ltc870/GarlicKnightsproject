User Stories for Sandra's Scenario 3

As the host of a book club, I want to be able to provide a solid book for the whole group.

As the host of a book club, I want a recommendation tool that has a wide range of books.

As the host of a book club, I want a recommendation tool that sends up to a purchase page.

As the host of a book club, I want a recommendation tool that has multiple forms of the same book, so that everyone in the club can purchase their preferred reading style.

As the host of a book club, I want to be able to leave reviews for others to see, as well as bea ble to read reviews of any recommendations.

As the host of a book club, I want to see if any given recommendation is a standalone book or if it is from a series.

As the host of a book club, I want to be able to share and see other peoples' book lists to pick out a proper selection for the book club.

As the host of a book club, I want to be able to compare all of our preferences and have a recommendation that appeals to all members of our club.
